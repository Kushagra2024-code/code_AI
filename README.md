# AI OA Practice

AI OA Practice is an open-source interview preparation platform for coding and system design, powered by AI and designed for free-tier deployment.

## Monorepo layout

- frontend/flutter_app: Flutter web/mobile app (Riverpod + Material UI)
- backend/cloud_functions: Firebase Cloud Functions APIs
- ai/genkit_flows: Genkit + Gemini AI pipelines
- compiler/judge_service: Judge0 integration proxy/service
- docs: architecture and deployment documentation

## Core capabilities

- AI coding interview simulator
- AI-generated coding questions
- Code execution with Judge0
- Hidden test case scoring
- AI code mentor feedback
- System design drawboard + AI evaluation
- Cheating detection signals
- Session scoring and analytics dashboard
